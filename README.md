# Wallet Dashboard Pro - GitHub Pages

Esta aplicación está lista para ser alojada en **GitHub Pages**. Todos los archivos están en la raíz para que el despliegue sea automático.

## Estructura de Carpetas
- `/assets/images`: Guarda aquí tus fotos, logos y fondos de tarjetas.
- `/assets/videos`: Para futuros contenidos multimedia pesados.
- `/assets/multimedia`: Recursos adicionales.
- `/utils`: Lógica extra que decidas separar del archivo principal.

## Cómo usar desde el Móvil
1. Sube estos archivos a tu repositorio de GitHub.
2. Activa **GitHub Pages** en los ajustes del repositorio (Settings -> Pages).
3. Selecciona la rama `main` y la carpeta `/root`.
4. ¡Tu billetera estará online!

## Nota sobre Multimedia
Para usar una imagen guardada en `/assets/images/foto.jpg` dentro de tu código, usa la ruta: `assets/images/foto.jpg`.
